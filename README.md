# Aksoy Park ve Kent Ekipmanları

Bu proje, Aksoy Park firması için hazırlanmış, React ve Vite kullanılarak geliştirilmiş kurumsal bir web sitesidir. 

## 🛠 Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyin:

1. Bağımlılıkları yükleyin:
   npm install

2. npm run dev

3. npm run build

4. npx wrangler pages deploy dist
